# Quick Start

Profile Signal Kit works with GitHub's real README renderer. The dark visual design lives inside local SVG files, not custom README CSS.

1. Create or open your special GitHub profile repository: `username/username`.
2. Copy `templates/PROFILE-README-TEMPLATE.md` into that repo as `README.md`.
3. Copy the assets you want from `assets/`.
4. Replace placeholders:
   - name
   - handle
   - links
   - project titles
   - descriptions
   - focus lines
   - working style lines
5. Commit and push to GitHub.
6. Check your public profile in GitHub light mode and dark mode.

Keep the order simple:

```text
Hero
Badges / typing line
Divider
Selected work
Current focus
Working style
Profile signals
Footer signal
```
