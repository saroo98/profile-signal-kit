## 6. README structure

Use:
- clear title
- concise intro
- feature list
- quick start
- screenshots or assets
- customization guide
- keywords in headings where natural

## 7. Alt text guidance

- Give meaningful alt text to hero banners, project cards, focus panels, and social preview images.
- Use empty alt text only for decorative dividers.
- Keep critical facts readable in Markdown or clear SVG text.

## 8. Backlinks and sharing

- Add this repository to relevant profile README lists only when allowed by their contribution rules.
- Share a before/after profile screenshot on your own profile, blog, or social account.
- Link back from profile READMEs that use the kit if you want to support discoverability.
